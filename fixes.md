# Troubleshooting

## Fixing misbehaving Java applications
Try setting export _JAVA_AWT_WM_NONREPARENTING=1. Also see the [Java](https://wiki.archlinux.org/index.php/Java#Gray_window,_applications_not_resizing_with_WM,_menus_immediately_closing) page.
